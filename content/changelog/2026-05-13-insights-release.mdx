export const title = "Inngest Insights: New datasources and features for querying your data";
export const date = "2026-05-13";

We've released Insights of out public beta and added new datasources and improved function and syntax support.

New datasources include:
* [Runs](/docs/platform/monitor/insights#runs-table): Query your function runs and their associated data, such as input, output, and execution time.
* [Steps/StepAttempts](/docs/platform/monitor/insights#steps-and-step-attempts-table): Query the individual steps within your function runs, including their status, output, and execution time.
* [Extended Trace Spans](/docs/platform/monitor/insights#extended-trace-spans-table): Query the spans associated with your function runs, providing deeper insights into the execution of your functions.

In addition to new datasources, we've also added support for [new functions](/docs/platform/monitor/insights#supported-functions) and improved syntax support to make it easier to query your data.
