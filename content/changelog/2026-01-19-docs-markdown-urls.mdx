export const title = "Docs markdown actions for LLMs";

Our docs now have quick actions to copy or open the current page in Markdown format. You can also open a new conversation with the markdown URL in Claude, ChatGPT or other LLM providers. The Markdown format reduces the amount of context needed to be sent to LLMs and coding agents.

<img src="/assets/changelog/2026-01-19-docs-markdown.png" alt="Docs markdown options" className="max-w-sm"/>

The [`llms.txt`](/llms.txt) file now also includes the markdown URLs for each page, reducing the required context
