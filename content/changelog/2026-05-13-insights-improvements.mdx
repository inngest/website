export const title = "Insights: New datasources and features for querying your data";
export const date = "2026-05-13";

We've improved Insights and added new datasources, functionality and syntax support.

New datasources include:
* [Runs](/docs/platform/monitor/insights#runs-table): Query your function runs and their associated data, such as input, output, and execution time.
* [Steps/StepAttempts](/docs/platform/monitor/insights#steps-and-step-attempts-table): Query the individual steps within your function runs, including their status, output, and execution time.
* [Extended Trace Spans](/docs/platform/monitor/insights#extended-trace-spans-table): Query the spans associated with your function runs, providing more information about the execution of your functions.

In addition to new datasources, we've also added support for [new functions](/docs/platform/monitor/insights#supported-functions) and [improved syntax](/docs/platform/monitor/insights#json-dot-notation-recommended) support to make it easier to query your data.

We've also improved the Insights sidebar with new features including:
 * [Insights AI](/docs/platform/monitor/insights#insights-ai) — An assistant to help write your queries from plain English descriptions.
 * [Schema Explorer](/docs/platform/monitor/insights#schema-explorer) — A sidebar utility that shows available tables and event schemas.

Check out the [Insights documentation](/docs/platform/monitor/insights) for more details on the new features.
