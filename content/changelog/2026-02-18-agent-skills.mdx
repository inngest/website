We've released [Inngest Agent Skills](/docs/ai-dev-tools/agent-skills) — pre-built skills for AI coding agents like Claude Code, Cursor, and Windsurf that provide up-to-date guidance on building with Inngest.

Six skills are available covering the full Inngest development workflow:

- **inngest-setup** — Project configuration and SDK installation
- **inngest-events** — Event design, naming conventions, and idempotency
- **inngest-durable-functions** — Function creation with triggers, retries, and error handling
- **inngest-steps** — Workflow composition with step methods
- **inngest-flow-control** — Concurrency, throttling, rate limiting, and batching
- **inngest-middleware** — Cross-cutting concerns and dependency injection

Pair with the [Dev Server MCP integration](/docs/ai-dev-tools/mcp) for a complete AI-powered write, test, and debug loop.

Learn more in the [Agent Skills documentation](/docs/ai-dev-tools/agent-skills) or check out the [inngest-skills repo on GitHub](https://github.com/inngest/inngest-skills).
