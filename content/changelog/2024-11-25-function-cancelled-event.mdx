export const title = "New inngest/function.cancelled event";
export const date = "2024-11-25";

A new system event has been added to allow you to write functions that run after other function runs are cancelled. This enables you to handle cancelled runs gracefully.

* [Example: Cleanup after function cancellation](/docs/examples/cleanup-after-function-cancellation)
* `inngest/function.cancelled` - [System event reference](/docs/reference/system-events/inngest-function-cancelled)
