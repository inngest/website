We've released Insights, a new feature that allows you to query your events with SQL directly in the Inngest dashboard. This initial release is focused on querying events, but support for querying runs and other data sources will be added in the future.

Read the [full blog post](/blog/insights-query-events-and-runs?ref=changelog-insights) for more details.
