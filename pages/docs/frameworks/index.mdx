# Frameworks

Inngest supports several frameworks out of the box. The set up flow for each framework is typically:

1. Add the `inngest` SDK to your project
2. Set up the `serve()` API handler
3. Create functions!

Check our our framework guides to learn how to get started:

- [Next.js](/docs/frameworks/nextjs)
- [RedwoodJS](/docs/frameworks/redwoodjs)
- [Express.js](/docs/frameworks/express)
- [Cloudflare Pages](/docs/frameworks/cloudflare-pages)

Would you like to see additional frameworks supported? [Drop us a note on our Discord community](/discord)
