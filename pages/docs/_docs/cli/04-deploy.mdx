The `inngest deploy` command is used deploy your function to the Inngest platform.

It uses Docker and the `Dockerfile` in your directory to create a conatiner image and push it to the Inngest private registry. You must be registered and [`login`](/docs/cli/login) to use this command.

## Usage

```
$ inngest deploy
```
