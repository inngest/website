The `inngest login` command is used authenticate the cli to manage your functions on the Inngest platform.

<aside>
  <p>
    An Inngest account is requried to login:{" "}
    <a href="/sign-up?ref=docs-cli-login">create one for free here</a>!
  </p>
</aside>

## Usage

```
$ inngest login
```

Or pass your credentials via arguments:

```
$ inngest login -u <your username> -p <your password>
```
