# Run

Run steps in parallel.

```py
@inngest.create_function(
    fn_id="my_function",
    trigger=inngest.TriggerEvent(event="app/my_function"),
)
async def fn(
    ctx: inngest.Context,
    step: inngest.Step,
) -> int:
    async def my_step() -> int:
        # Do stuff
        return 1

    return await step.run("my_step", my_step)
```
